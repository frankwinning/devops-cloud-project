# Day 4 — System Monitoring Automation

## Script: system_monitoring.sh

### Purpose
Automates system resource monitoring to check CPU, memory, disk I/O, and processes.

### Tools Used
- top/htop → Process monitoring
- vmstat → Virtual memory stats
- iostat → Disk I/O statistics

### How It Helps
- Quickly identify resource bottlenecks
- Useful for server health checks in DevOps pipelines
- Provides data to optimize performance
