## Script: system_health_check.sh

### Purpose
Automates basic Linux system health checks.

### What It Does
- Shows hostname, current user, date
- Displays uptime
- Shows disk usage
- Shows memory usage
- Lists top 5 processes by memory

### Why It Matters
Automating system checks saves time and avoids human error.  
This is a common task for DevOps engineers monitoring servers.
